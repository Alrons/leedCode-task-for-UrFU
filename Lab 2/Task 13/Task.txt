В городе Дигитвилль существовал список чисел, называемый numsцелыми числами от 0до n - 1. Каждое число должно было встречаться в списке ровно один раз , однако два проказливых числа прокрались в список ещё один раз , сделав список длиннее обычного.

Ваша задача, как городского детектива, — найти эти два хитрых числа. Верните массив размером два, содержащий эти два числа (в любом порядке ), чтобы мир вернулся в Диджитвилль.

 

Пример 1:

Ввод: числа = [0,1,1,0]

Выход: [0,1]

Объяснение:

Числа 0 и 1 встречаются в массиве дважды.

Пример 2:

Ввод: числа = [0,3,2,1,3,2]

Выход: [2,3]

Объяснение:

Числа 2 и 3 встречаются в массиве дважды.

Пример 3:

Ввод: числа = [7,1,5,4,3,4,6,0,9,5,8,2]

Выход: [4,5]

Объяснение:

Числа 4 и 5 встречаются в массиве дважды.

 

Ограничения:

2 <= n <= 100
nums.length == n + 2
0 <= nums[i] < n
Входные данные генерируются таким образом, чтобы numsони содержали ровно два повторяющихся элемента.
