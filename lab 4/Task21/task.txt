Если задано целое число rowIndex, вернуть строку ( с индексом 0 ) треугольника Паскаля .rowIndexth

В треугольнике Паскаля каждое число представляет собой сумму двух чисел, расположенных непосредственно над ним, как показано:


 

Пример 1:

Вход: rowIndex = 3
 Выход: [1,3,3,1]
Пример 2:

Вход: rowIndex = 0
 Выход: [1]
Пример 3:

Вход: rowIndex = 1
 Выход: [1,1]
 

Ограничения:

0 <= rowIndex <= 33
 

Дополнение: Можете ли вы оптимизировать свой алгоритм так, чтобы он использовал только O(rowIndex)дополнительное пространство?
